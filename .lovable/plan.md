Quyết định kiến trúc (đã chọn tối ưu)\n\n- **Frontend**: Web app React/TanStack Start (responsive mobile, demo trên cả laptop & điện thoại) — thay cho Flutter.\n- **Backend model**: **FastAPI tự host** (Docker) trên laptop của nhóm, expose qua **ngrok / cloudflared tunnel** → có URL HTTPS public thật → Pi và web đều truy cập được, không cần mở port router. Đây là cách thực tế nhất cho demo PBL5 (không tốn tiền GPU cloud, vẫn truy cập được từ ngoài).\n- **DB + Auth + lịch sử user**: **Lovable Cloud (Supabase)** — không phải tự dựng Postgres, có sẵn Auth, RLS, dashboard.\n- **Raspberry Pi**: **Stream JPEG frames** qua WebSocket (15 FPS, resize 640×360, JPEG quality 70) → backend chạy MediaPipe + model. Lý do: (1) đồng bộ 100% với `live_test.py` hiện tại, không phải port MediaPipe sang Pi (Pi 4 chạy MediaPipe Holistic chỉ ~5 FPS), (2) dễ debug, (3) băng thông LAN/4G dư sức (~300 KB/s).\n\n## 3. Kiến trúc tổng thể\n\n```text\n┌──────────────┐   WSS (JPEG)    ┌────────────────────┐\n│ Raspberry Pi │ ───────────────▶│  FastAPI Inference │\n│  + Camera    │◀── ack/result ──│  (MediaPipe +      │\n└──────────────┘                 │   BiLSTM+Attention)│\n                                 └─────────┬──────────┘\n                                           │ REST + WSS\n                                           ▼\n┌────────────────┐   HTTPS    ┌────────────────────────┐\n│  Web App       │◀──────────▶│  Lovable Cloud         │\n│  (React/TSS)   │            │  (Auth, DB, Storage)   │\n│  - Live view   │            │  - users, sessions,    │\n│  - Học SL      │            │    predictions,        │\n│  - Lịch sử     │            │    lessons, attempts   │\n└────────────────┘            └────────────────────────┘\n```\n\nPi cũng có thể stream thẳng tới Web (peer mode) qua FastAPI relay — nhưng mặc định: Pi → FastAPI → Web (FastAPI là hub, fan-out frame + result tới mọi web client của user đó).\n\n## 4. Tính năng phần mềm\n\n### A. Core (bắt buộc)\n1. **Đăng ký / Đăng nhập** (email + password, qua Lovable Cloud Auth).\n2. **Live Recognition**:\n   - Trang hiển thị video stream từ Pi (MJPEG hoặc canvas vẽ frame qua WS).\n   - Overlay landmarks (pose/hand/face) — vẽ ở client để giảm tải.\n   - Panel kết quả: từ dự đoán + confidence + top-3 + thanh progress.\n   - Nút **Start/Stop session**, lưu session + danh sách từ đã nhận → DB.\n3. **Lịch sử phiên**: list session (thời gian, số từ, accuracy trung bình), click vào xem chi tiết từng từ + timestamp.\n4. **Module học ngôn ngữ ký hiệu**:\n   - Danh sách 100 từ vựng (theo `100_index_mapping.json`), mỗi từ có **video mẫu** + mô tả.\n   - **Practice mode**: app hiện 1 từ → user ký hiệu trước camera → backend chấm xác suất → trả \
